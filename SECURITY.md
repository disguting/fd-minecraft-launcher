# Security Policy

## Supported Versions

The following versions of the Minecraft Launcher are currently receiving security updates. We strongly recommend always using the latest stable release.

| Version | Supported          |
| ------- | ------------------ |
| 1.5.1   | :white_check_mark: |

## Reporting a Vulnerability

We take the security of our launcher and our users' data very seriously. Since the launcher handles sensitive information (such as authentication tokens for license accounts or offline profile data), please **do not** report security vulnerabilities through public GitHub issues.
