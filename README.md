# Minecraft Launcher (C#)

A lightweight, open-source Minecraft launcher built with C# and .NET, focused on simplicity, performance, and transparency.

---

## 🚀 Why this launcher?

Most Minecraft launchers try to do everything. This one focuses on doing the essentials **fast and clean**.

* ⚡ Lightweight and fast startup
* 🧩 Built-in Modrinth integration
* 🗂 Clean instance and profile management
* 🎯 Minimal UI without unnecessary complexity
* 🔓 Fully open-source and transparent

If you want a simple launcher without bloat, this is for you.

## ✨ Features

* **Microsoft Authentication** (official OAuth flow)
* **Offline Mode** support
* **Mod Integration** (Forge & Fabric mods support)
* **Instance Management** (separate directories for each setup)
* **Fast Launching** with optimized resource handling

## 🔐 Security & Privacy

This project is built with transparency in mind.

* ❌ No telemetry
* ❌ No analytics
* ❌ No data collection
* ✅ Only official Microsoft authentication
* ✅ Requests only to Microsoft and Modrinth APIs
* ✅ No credentials stored in plain text
* 🔍 Fully open-source — you can verify everything yourself

## 🛠 Installation

### Requirements

* Windows 10/11
* Java Runtime Environment (JRE)
* .NET Runtime (if not bundled)

### Build from source

1. Clone the repository:

   ```
   git clone https://github.com/disguting/fd-minecraft-launcher
   ```

2. Open the solution in Visual Studio 2022

3. Restore NuGet packages

4. Build in Release mode

---

## ▶️ Usage

1. Launch the application
2. Log in using Microsoft or use offline mode
3. Select a game version
4. Click **Play**
