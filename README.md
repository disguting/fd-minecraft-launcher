# Minecraft Launcher (C#)

A custom-built Minecraft launcher developed in C# using the .NET framework. This project is designed to provide a streamlined experience for managing game versions, mods, and user profiles.

## Features
* **Authentication:** Support for both Microsoft accounts (license) and offline mode.
* **Mod Integration:** Built-in search and installation tools for Forge and Fabric environments.
* **Profile Management:** Independent directory structures for different game instances and modpacks.
* **Performance:** Optimized resource handling for fast loading and game launching.

## Technical Stack
* **Language:** C#
* **Framework:** .NET 10.0
* **UI Framework:** WPF (Windows Presentation Foundation)
* **License:** GNU GPL v3.0

## Installation and Build
You can download the latest installer here:
   https://github.com/disguting/fd-minecraft-launcher/releases

To build this project locally, ensure you have Visual Studio 2022 and the .NET SDK installed.

1. Clone the repository:
   git clone https://github.com/disguting/fd-minecraft-launcher
2. Open the solution file (.sln) in Visual Studio.
3. Restore NuGet packages.
4. Build the solution in Release mode or press F5 to debug.

## Usage
Upon launching, select your preferred version and click the Play button. Ensure you have the appropriate Java Runtime Environment (JRE) installed on your system to run the game.
